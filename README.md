1\. Types of Data

(a) Numerical Data

Numerical data represents numbers that can be measured or counted. Mathematical operations like addition, subtraction, mean etc. can be performed.

Examples from dataset:

Household\_Income = 50000
Age\_Of\_Household\_Head = 45
Family\_Size = 5

Types:
Discrete: Countable numbers (Family\_Size)
Continuous: Measurable values (Income, Age)

(b) Categorical Data
Categorical data represents labels or categories and cannot be used for mathematical calculations.

Examples from dataset:

Education\_Level = Graduate
Owns\_House = Yes / No
Urban\_Rural = Urban / Rural

Types:

Nominal: No order (Urban/Rural)
Ordinal: Ordered categories (Primary, Secondary, Graduate)


2\. Types of Statistics


(a) Descriptive Statistics

Descriptive statistics summarize and describe the main features of a dataset.

Common tools:
Mean
Median
Mode
Standard Deviation
Graphs \& Charts

Example:
Average Household Income of 150 households.


(b) Inferential Statistics
Inferential statistics uses sample data to make predictions or conclusions about a population.

Methods:
Hypothesis Testing
Confidence Intervals
Regression Analysis

Example:

Predicting the average income of all households in a city using a sample dataset.


3\. What is Descriptive Statistics?

Descriptive statistics is a method used to organize, summarize, and present data in meaningful ways using numbers, tables, or graphs.

Common Measures:

Mean
Median
Mode
Range
Standard Deviation


Example:
If incomes are:
30000, 40000, 50000, 60000

Mean income = 45000

4\. Percentiles

Percentiles divide data into 100 equal parts.

Example:

25th percentile → 25% data below this value
90th percentile → 90% data below

Used in:
Exam rankings
Income analysis
Formula
P = (n / 100) × k

5\. Quartiles


Quartiles divide data into 4 equal parts.

Quartile	Meaning
Q1	25% of data
Q2	Median (50%)
Q3	75% of data

8\. Kurtosis
Kurtosis measures peakedness of distribution.

Types:

Leptokurtic – very sharp peak
Mesokurtic – normal peak
Platykurtic – flat peak
Formula
Kurtosis = E\[(X − μ)⁴] / σ⁴
